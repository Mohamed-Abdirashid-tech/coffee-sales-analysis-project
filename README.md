# ☕ Coffee Sales Analysis Project

## 📌 Project Overview

This project focuses on analyzing coffee sales data using Python and Jupyter Notebook. The goal of this analysis is to transform raw sales data into meaningful business insights through data cleaning, exploratory data analysis (EDA), visualization, and business intelligence techniques.

This project demonstrates practical data analytics skills and is designed as a professional portfolio project for GitHub.

---

# 🎯 Project Objectives

The main objectives of this project are:

- Clean and preprocess raw sales data
- Analyze sales performance and profitability
- Identify top-performing coffee products
- Discover sales trends and customer behavior
- Create professional data visualizations
- Generate actionable business insights
- Support data-driven business decisions

---
